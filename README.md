# Basic Rotary-Encoder-Debugging-and-Troubleshooting
This code is helpful for a quick debugging of the rotary encoder that has SW, CLK, and DT pins, 
it uses basic coding strategies to help troubleshoot the rotary encoder. The user will know
both whether there is a change in the direction of rotation or not and the direction either
ClockWise, Anti-ClockWise rotation, or stale (no rotation, remaining idle)
